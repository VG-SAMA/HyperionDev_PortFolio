{% endblock %} {% comment %} note_detail.html (Note Detail Page) Purpose: -------- This template displays the details of a single note in the Sticky Notes app. It shows the note's title, content, and creation date, and provides action buttons for updating, deleting, or returning to the notes list. Features: --------- - Extends base.html to inherit the common header, layout, and styling. - Displays note details in a centered Bootstrap card for clean UI. - Includes action buttons: - Update: Redirects to the edit form for the note. - Delete: Redirects to the confirmation delete page. - Back: Returns to the main notes list. - Uses Bootstrap classes for responsive design and layout. - Uses Bootstrap Icons for buttons (pencil, trash, arrow). Template Blocks: ---------------- - title: Sets the browser tab title dynamically with the note's title. - header: Displays the note title as the page header. - content: Contains the card displaying the note details and action buttons. Included Libraries: ------------------- - Bootstrap 5 for responsive layout and styling. - Bootstrap Icons for visual icons. - Inherits custom CSS from base.html (notes/styles.css). Usage: ------ This template is rendered by the `get_note` view in views.py, passing the selected note object as context. Reference: https://getbootstrap.com/docs/4.4/components/buttons/ Icons: https://icons.getbootstrap.com/?q=edit {% endcomment %}
